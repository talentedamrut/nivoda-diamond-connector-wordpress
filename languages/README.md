# Nivoda API Integration - Languages

This directory contains translation files for the Nivoda API Integration plugin.

## Available Translations

Currently no translations are available. Translation files (.po and .mo) should be placed in this directory.

## Contributing Translations

To contribute translations:

1. Generate the POT file using WP-CLI or Poedit
2. Create language-specific .po files 
3. Compile to .mo files
4. Submit via WordPress.org translation system

## Text Domain

The plugin uses the text domain: `nivoda-api-integration`
