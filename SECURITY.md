# Security Policy

## Supported Versions
The latest major release of this plugin receives security updates.

## Reporting a Vulnerability
If you discover a security issue, please contact us directly:

**collab@amrutratnani.com**

Please do not create a public GitHub issue for security reports.

We will:
- Confirm the issue  
- Provide a fix  
- Announce the resolution (without exposing sensitive details)
