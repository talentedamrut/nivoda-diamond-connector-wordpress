# Contributing to Nivoda Diamond Shop

Thank you for considering contributing!

## How to Contribute

### 1. Fork the repository
Click **Fork** and work on your own copy.

### 2. Create a new branch

### 3. Make your changes
Follow WordPress and PHP coding standards.

### 4. Commit with a clear message

### 5. Push and open a Pull Request
Explain what you changed and why.

## Reporting Issues
Please include:
- Steps to reproduce  
- Expected behavior  
- Actual behavior  
- Screenshots or logs  

## Code Style
- Follow WordPress Coding Standards (WPCS)  
- Use meaningful variable names  
- Avoid long functions (split into smaller ones)  

## Thank You
Your contributions help this plugin grow and benefit the community!
